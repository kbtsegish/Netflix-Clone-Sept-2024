# Netflix Clone

This is a responsive clone of the Netflix homepage, built using HTML, CSS, and JavaScript.

## Features
- Responsive design for mobile and tablet
- CSS Flexbox and Grid for layout
- Hover animations
- Video previews

- CSS3
- JavaScript
